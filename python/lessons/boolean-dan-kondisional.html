Boolean & Kondisional

Kenali tipe bool, operator perbandingan, logika and/or/not, serta penerapan kondisional untuk mengendalikan alur program.

Boolean & Tipe bool

Python menyediakan tipe bool dengan dua nilai saja: True dan False. Biasanya kita mendapat nilainya dari hasil perbandingan.

x = True
print(x)
print(type(x))

Operator Perbandingan

Operasi Deskripsi Operasi Deskripsi
a == ba sama dengan ba != ba tidak sama dengan b
a < ba kurang dari ba > ba lebih dari b
a <= ba kurang/sama dengan ba >= ba lebih/sama dengan b
 
def can_run_for_president(age):
    """Apakah seseorang boleh mencalonkan diri sebagai presiden AS?"""
    return age >= 35

print(can_run_for_president(19))
print(can_run_for_president(45))

Gunakan == ketika membandingkan nilai; = hanya untuk assignment.

Perbandingan + Aritmetika

Operator perbandingan dapat digabung dengan operasi matematika. Contohnya untuk memeriksa bilangan ganjil:

def is_odd(n):
    return (n % 2) == 1

print(is_odd(100))
print(is_odd(-1))

Menggabungkan Boolean

Operator and, or, dan not menggabungkan ekspresi boolean.

def can_run_for_president(age, is_natural_born_citizen):
    return is_natural_born_citizen and (age >= 35)

print(can_run_for_president(19, True))
print(can_run_for_president(55, False))
print(can_run_for_president(55, True))

and memiliki prioritas lebih tinggi daripada or. Tambahkan tanda kurung demi keterbacaan dan mencegah bug.

prepared_for_weather = (
    have_umbrella
    or ((rain_level < 5) and have_hood)
    or (not (rain_level > 0 and is_workday))
)

Kondisional

Boolean menjadi dasar if, elif, dan else yang menentukan cabang eksekusi.

def inspect(x):
    if x == 0:
        print(x, "adalah nol")
    elif x > 0:
        print(x, "positif")
    elif x < 0:
        print(x, "negatif")
    else:
        print("Tidak terdefinisi")

inspect(0)
inspect(-15)

Perhatikan titik dua dan indentasi empat spasi yang menandai blok kode.

Blok & Efek Samping

def f(x):
    if x > 0:
        print("Hanya saat x positif", x)
    print("Selalu dicetak", x)

f(1)
f(0)

Baris di luar blok if selalu dieksekusi, apa pun nilai kondisinya.

Konversi ke Boolean

bool() mengubah tipe lain menjadi boolean. Angka non-nol atau string non-kosong dianggap truthy.

print(bool(1))
print(bool(0))
print(bool("asf"))
print(bool(""))

if 0:
    print(0)
elif "spam":
    print("spam")

Kode di atas mencetak "spam" karena string non-kosong otomatis bernilai True.
