Fungsi & Bantuan

Di modul ini kita memperdalam cara memakai fungsi bawaan, memanfaatkan dokumentasi interaktif, mendefinisikan fungsi sendiri, menulis docstring, hingga memberikan parameter opsional.

Mengenal Fungsi dan Bantuan

Fungsi seperti print() atau abs() sudah kita pakai sejak pelajaran pertama. Untuk memahami fungsi lain, ingat satu kunci penting: gunakan help().

help(round)

Output help() menampilkan header fungsi (mis. round(number, ndigits=None)) dan deskripsi singkat. Pastikan kamu memanggil help() dengan nama fungsi, bukan hasil panggilannyaâ€”help(round(-2.01)) justru memberi informasi tentang tipe integer yang dikembalikan.

Memahami Parameter

Dokumentasi print() memberi gambaran parameter opsional seperti sep, end, file, dan flush. Contoh:

print(1, 2, 3, sep=' < ')
print(1, 2, 3)

sep default-nya spasi, sehingga jika tidak diisi, hasilnya tetap 1 2 3.

Mendefinisikan Fungsi

Gunakan kata kunci def untuk membuat fungsi baru. Contoh berikut mencari selisih terkecil dari tiga angka:

def least_difference(a, b, c):
    diff1 = abs(a - b)
    diff2 = abs(b - c)
    diff3 = abs(a - c)
    return min(diff1, diff2, diff3)

print(
    least_difference(1, 10, 100),
    least_difference(1, 10, 10),
    least_difference(5, 6, 7),
)

return menghentikan fungsi dan mengirim nilai kembali ke pemanggil.

Docstring

Tambahkan deskripsi fungsi menggunakan triple quote tepat setelah header. Docstring akan muncul ketika help() dipanggil.

def least_difference(a, b, c):
    """Kembalikan selisih terkecil di antara tiga angka.

    >>> least_difference(1, 5, -5)
    4
    """
    diff1 = abs(a - b)
    diff2 = abs(b - c)
    diff3 = abs(a - c)
    return min(diff1, diff2, diff3)

Sertakan contoh pemanggilan untuk membantu pembaca memahami ekspektasi fungsi.

Fungsi Tanpa return

Jika kita lupa menulis return, Python tetap mengizinkan fungsi berjalan, tetapi hasil pemanggilannya menjadi None.

def least_difference(a, b, c):
    diff1 = abs(a - b)
    diff2 = abs(b - c)
    diff3 = abs(a - c)
    min(diff1, diff2, diff3)

print(least_difference(1, 10, 100))  # None

Beberapa fungsi memang sengaja tidak mengembalikan nilai karena hanya membutuhkan efek samping, misalnya print() dan help().

Argumen Default

Tambahkan nilai default agar pengguna tidak wajib mengisi setiap parameter.

def greet(who="Colin"):
    print("Hello,", who)

greet()
greet(who="Kaggle")
greet("world")

Parameter default sangat mirip dengan perilaku print() yang memiliki sep bawaan.

Fungsi sebagai Argumen

Python mendukung higher-order function, yakni fungsi yang menerima fungsi lain sebagai input.

def mult_by_five(x):
    return 5 * x

def call(fn, arg):
    return fn(arg)

def squared_call(fn, arg):
    return fn(fn(arg))

print(call(mult_by_five, 1))
print(squared_call(mult_by_five, 1))

Kamu tidak perlu langsung menulis fungsi seperti ini, tetapi berguna untuk memahami konsepnya karena banyak library Python memanfaatkannya.

Parameter key pada max()

Kita bisa mengirim fungsi pembanding ke argumen key. Contoh berikut mencari angka terbesar berdasarkan sisa bagi 5.

def mod_5(x):
    return x % 5

print(
    'Which number is biggest?',
    max(100, 51, 14),
    'Which number is the biggest modulo 5?',
    max(100, 51, 14, key=mod_5),
    sep='\n',
)

Hasilnya 14 untuk versi modulo karena 14 menyisakan 4, nilai terbesar dibandingkan sisa angka lainnya.
